    }

    }

        读者ID
        读者类型
    }

    }

        书号
        书名
        作者
        出版社

        ISBN

        预订数量
    }

        图书编号

    }
    读者 <|-down-老师
    读者 <|-right- 学生
    借阅记录 "0..5" -down-"1" 读者:借阅
    
    图书管理系统 "1"-right-"0..*"读者
    读者 "0..1" -left- "0..3" 预订记录:预订
    借阅记录 "0..n" -left- "1" 馆员:管理;
    馆员 "1..*" -left-  "0..*"图书:管理

    图书管理系统 -down- 罚款:查询
    图书管理系统 "1"-left-"1" 书目:拥有
    维护人员 "1..*" -down- "1" 图书管理系统:维护
    预订记录 "0..n" -left-  "0..1"书目:预订
    图书 "n" -up-  "1" 书目
